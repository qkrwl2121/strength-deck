# Strength Deck

Mobile-first PWA for strength training planning, 1RM-based prescriptions, local workout history, and local data export/import.

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to Settings -> Pages.
4. Set Source to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Open the generated `https://<username>.github.io/<repo>/` URL in iPhone Safari.
7. Use Share -> Add to Home Screen.

User data is stored locally in the browser/PWA storage for the same deployed URL. Use My Page -> Data Export for backup.
